# XYZ Company Sales Dashboard

## Overview
This project presents an interactive e-commerce sales dashboard designed to analyze sales performance, profitability, order volume, and customer behavior. The dashboard enables users to explore business performance across different quarters and product dimensions.

## Dashboard Features
- Quarter-based slicer (Q1 to Q4) for seasonal analysis
- Top 4 selection slicer for focused performance review
- Dynamic KPIs reflecting slicer selections

## Key KPIs
- Total Sales Amount
- Total Profit
- Total Quantity Sold
- Average Order Value (AOV)

## Key Visuals
- Sales amount by state
- Quantity sold by product category
- Top customers by sales contribution
- Monthly profit trend
- Profit by product category

## Data Analysis
The dashboard analyzes transactional e-commerce data to evaluate regional performance, category-wise demand, customer contribution, and profit trends over time. Ranking and aggregation techniques are used to highlight top-performing segments.

## Tools Used
- Power BI
- DAX
- Data modeling and visualization
- Interactive slicers and KPI analysis

## Purpose
This dashboard supports data-driven decision-making by providing clear insights into sales efficiency, profitability, customer performance, and product trends.
